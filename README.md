## Note for creating one

* Using Markdowns built-in link synatxes do not work with Data URIs
* The Data URI cannot handle big HTML by using `data:text/html,<html-code-here>` so instead encode the html in base64 and then do `data:text/html;base64,<base64-code-here>`
